## 📇 What is VCF?

**VCF** (Virtual Contact File) or **vCard** is the standard file format for electronic business cards. It stores contact information like names, phone numbers, emails, and addresses — readable by all phones and email clients.

| Format | Extension | MIME Type |
|--------|-----------|-----------|
| vCard 3.0 | `.vcf` | `text/vcard` |
